1_Reactive
	Contém os scripts utilizados para os agentes Drone e Guarda,
	implementando uma arquitectura puramente reactiva.

2_Deliberative
	Contém os scripts utilizados para os agentes Drone e Guarda,
	implementando uma arquitectura deliberativa (BDI).

3_Hybrid_with_cooperation
	Contém os scripts utilizados para os agentes Drone e Guarda,
	implementando uma arquitectura híbrida.
	Além disto, estes scripts implementam também a componente
	de comunicação e cooperação entre agentes.
